Endpoints Disponíveis

Aluno

  • POST /api/alunos: Adiciona um novo aluno.

    Para adicionar um novo aluno, envie uma requisição POST para a rota /api/alunos com os dados do aluno no corpo da requisição em formato JSON.

  • GET /api/alunos: Retorna todos os alunos cadastrados.

    Para obter todos os alunos cadastrados, envie uma requisição GET para a rota /api/alunos. A resposta será uma lista de objetos JSON representando os alunos.

  • GET /api/alunos/:id: Retorna um aluno específico pelo ID.

    Para obter um aluno específico pelo seu ID, envie uma requisição GET para a rota /api/alunos/:id, substituindo :id pelo ID do aluno desejado. A resposta será um objeto JSON representando o aluno encontrado.

  • DELETE /api/professores/:id: Remove um aluno pelo ID.

    Para remover um aluno pelo seu ID, envie uma requisição DELETE para a rota /api/professores/:id, substituindo :id pelo ID do aluno a ser removido. Se o aluno for encontrado e removido com sucesso, a resposta será um status 204 No Content. Caso contrário, será retornado um status 404 Not Found.

Aula

  • POST /api/aulas: Adiciona uma nova aula.

    Para adicionar uma nova aula, envie uma requisição POST para a rota /api/aulas com os dados da aula no corpo da requisição em formato JSON.

  • GET /api/aulas: Retorna todas as aulas cadastradas.

    Para obter todas as aulas cadastradas, envie uma requisição GET para a rota /api/aulas. A resposta será uma lista de objetos JSON representando as aulas.

  • GET /api/aulas/:id: Retorna uma aula específica pelo ID.

    Para obter uma aula específica pelo seu ID, envie uma requisição GET para a rota /api/aulas/:id, substituindo :id pelo ID da aula desejada. A resposta será um objeto JSON representando a aula encontrada.

  • DELETE /api/aulas/:id: Remove uma aula pelo ID.

    Para remover uma aula pelo seu ID, envie uma requisição DELETE para a rota /api/aulas/:id, substituindo :id pelo ID da aula a ser removida. Se a aula for encontrada e removida com sucesso, a resposta será um status 204 No Content. Caso contrário, será retornado um status 404 Not Found.

  • POST /api/aulas/:id/matricularAluno/:alunoId: Matricula um aluno em uma aula específica.

    Para matricular um aluno em uma aula específica, envie uma requisição POST para a rota /api/aulas/:id/matricularAluno/:alunoId, substituindo :id pelo ID da aula e :alunoId pelo ID do aluno. Envie os dados do aluno no corpo da requisição em formato JSON. A resposta será um status 201 Created se o aluno for matriculado com sucesso, ou um status 404 Not Found se o aluno não for encontrado.

  • POST /api/aulas/:id/definirProfessor/:professorId: Define um professor para uma aula específica.

    Para definir um professor para uma aula específica, envie uma requisição POST para a rota /api/aulas/:id/definirProfessor/:professorId, substituindo :id pelo ID da aula e :professorId pelo ID do professor. Envie os dados do professor no corpo da requisição em formato JSON. A resposta será um status 201 Created se o professor for definido com sucesso, ou um status 404 Not Found se o professor não for encontrado.
